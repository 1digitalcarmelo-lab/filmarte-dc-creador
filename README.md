## Para actualizar tu carpeta actual

Copiá y reemplazá:
- index.html
- styles.css
- script.js
- config.js

NO reemplaces la carpeta assets si ya tiene tus fotos y música.

## Importante sobre config.js

Si ya habías editado mucho tu config.js, podés conservarlo, pero para que aparezca el poema tenés que agregar los bloques:
- poema
- frasesExtra

Esta versión ya los trae listos.
